Application Menus

Main Menu
  • File-->Exit - exit HPB Signals application
  • View-->IB Sessions - open IB sessions window (see below)
  • View-->C2 Responses - open C2 responses log window (see below)
  • Database-->Backup DB - backup database (binary backup) into HPBSignals installation directory backup subfolder (timestamp name)
  • Database-->Export Data - export database tables (csv text export) into HPBSignals installation directory backup subfolder (timestamp name)
  • Database-->Reset DB - reset database and clear all data, should be used with caution!
  • Help-->Help Contents - open online documentation page in a new browser window
  • Help-->About - information about HPBSignals application (see below)
