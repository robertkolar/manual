Toolbars

Main Toolbar
  • IB account - display and/or select an IB account (FA subaccount) for which IB orders are getting processed. The selection is available for Financial Advisor accounts.
  • Listen IB - detect and process IB orders (checked) or disable listening and processing (unchecked)
  • Publish C2 - convert IB orders to C2 signals and publish them to Collective2 (checked) or disable conversion (unchecked)
  • Delay C2 - instruct C2 to delay signals on C2 side for 30 seconds in order to group several signals into one C2 email
  • Price Updt - detect IB order price update and enable C2 signal replacement by submitting a new signal (and cancel existing) with the updated price
  • STK, FUT, OPT, FX - detect and process stock orders, futures orders, option orders, forex orders
  • Allow Rvrs - allow simple reversal orders with no dependent child orders. A reversal order gets converted into two C2 signals. The first one is a closing signal and the second one is an opening signal that is conditional upon the first signal. On C2 side a position must be closed first before it can be reversed.
  • Scaling - enable or disable order quantity scaling in the range between 1/100 (1%) and 100x (10000%). If enabled a C2 signal will have a different (upscaled or downscaled) quantity from the matching IB order.
  • Split View - display both IB Orders and C2 Signals on the same screen or on separate tabs
IB Orders and C2 Signals Toolbars

  • Undl - check which underlyings to display in a tab

  • Filter C2 signal entries according to signals status - working, cancelled, filled, expired, unknown
  • Export table content into CSV file
  • Refresh page
  • Sort entries by clicking table column header
