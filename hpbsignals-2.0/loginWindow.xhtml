Login Window

Collective2 login parameters:

  • C2 Email: enter your email address that you use with your Collective2 account
  • C2 Password: enter your Collective2 password
  • System: after entering the login info click System... button to retrieve available systems from the C2 site. If you own multiple systems, choose one of them. If systems are successfully retrieved, systems combo box turns green, otherwise an error message is displayed.

IB Trader Workstation parameters:

  • IB Host: IP address or hostname of the computer that IB is running on; if it is running on the same computer as HPB signals, you can leave the default setting (localhost)
  • IB Port: a socket connection port that IB is listening on. Configuration on IB side: Configure-->API-->All API Settings-->Socket port
  • Connect: after entering all parameters click Connect button to connect to IB. Accept the connection on IB side if needed. If the connection is successful, the text box turns green, otherwise an error message is displayed.

After a successful retrieval of C2 system(s) and IB connection establishment, click Start button to start the main application.
