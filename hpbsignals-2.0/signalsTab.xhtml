C2 Signals Tab

Fields in the C2 Signals tab:

  • #: table row number
  • Signal ID: C2 Signal ID returned by the C2 after a successful signal submission
  • Parent ID: C2 parent signal ID in case of a conditional order - a child order is submitted after the parent order is executed
  • Perm ID: a matching (unique) IB order ID that the signal is attached to
  • Action: BTO - Buy To Open, STO - Sell To Open, BTC - Buy To Close, STC - Sell To Close
  • Quant: quantity of the instrument to buy or sell
  • RQ: reversal quantity that is submitted in a dependent (child) C2 signal
  • RType: reversal type - parent, child or none
  • Undl: underlying of the contract traded
  • Symbol: symbol of the contract traded
  • Instrm: instrument - stock, option, future, forex
  • LMT: a limit price in case of a limit order
  • STP: a stop price in case of a stop order
  • TIF: Time In Force; GTC - Good Till Cancelled, DAY - day
  • OCA Group: One Cancels All order group, if one order in the group is executed, all others are getting cancelled
  • Acknowledged On: C2 signal creation date and time
  • Poll: a number of all/failed signal status C2 server responses
  • Fill Prc: C2 signal fill price
  • C2 signal status as reported by the Collective2 server: working, filled, cancelled, expired, unknown , click on the status to open up a C2 signal and IB to C2 conversion details window
