Database Backup

  • HPB Signals includes embedded SQL database for keeping track of IB orders, conversion statuses, IB sessions, C2 signals and C2 signal statuses
  • It is recommended to backup the database regularly, since it contains all history info and the current order/signal statuses
  • Database backup is triggered by clicking Database-->Backup DB menu item
  • During the backup procedure a progressbar (placed in the status bar) is being displayed
  • After completion an informational message window pops up
