Status Bar


The following status fields are displayed at the bottom of the application window:
  • IB Connection: connected to host:port (green) or Disconnected (red)
  • Connected On: date and time of the current IB connection or N/A if currently not connected
  • Services:
    1. IB to C2 Processor: alternate green - running
    2. Signal Status Poller: alternate green - running
    3. IB Connection Poller: alternate green - running
    4. Table Updater: alternate green - running
  • Progress Bar
  • Current time
