IB Orders Tab

Fields in the IB Orders tab:

  • #: table row number
  • Perm ID: unique IB order ID - unique across IB sessions
  • Order: IB order ID - not unique across IB sessions
  • Parent: parent IB order ID if one exists
  • Account: IB account for which orders are being processed
  • Cl: 3rd party IB client ID that order was originated by (0 for IB originated order)
  • C2 Signal: a matching C2 signal ID after IB to C2 conversion is finished and a signal is published to Collective2
  • Conv: IB order to C2 signal conversion status:
    • SBM - SubmitNew, SBM - SubmitProcessed, SBM - SubmitIgnored
    • UPD - UpdateNew, UPD - UpdateProcessed, UPD - UpdateIgnored
    • CNC - CancelNew, CNC - CancelProcessed, CNC - CancelIgnored
    Click on the status to open up IB order and IB to C2 conversion details window.
  • Retries: number of retransmissions of a signal/update/cancel request and a wait timer in case of a failure condition
  • Action: BUY or SELL
  • Quant: quantity of the contract to buy or sell
  • Undl: underlying of the contract traded
  • Symbol: symbol of the contract traded
  • Instrm: instrument - stock, option, future, forex
  • Type: order type - limit (LMT) or stop (STP)
  • Sbm Prc: submit price - a limit price in case of a limit order or a stop price in case of a stop order
  • TIF: Time In Force; GTC - Good Till Cancelled, DAY - day
  • OCA Group: One Cancels All order group, if one order in the group is executed, all others are getting cancelled
  • Submitted On: IB order submission date and time
  • Fill Prc: IB order fill price
  • Status as reported by IB: submitted, updated, filled, cancelled, unknown , click the status column to open up IB order and IB to C2 conversion details window
