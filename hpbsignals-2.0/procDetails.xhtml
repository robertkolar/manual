Processing Details and Events

IB order events and IB to C2 conversion events
  • Opened by clicking the conversion or status column of an IB order row
  • A history log of the selected IB order status transitions is displayed: Event Date, Status, Submit Price, Fill Price
  • A history log of the IB to C2 conversion status transitions is displayed: Event Date, Status, C2 Signal, Description
C2 signal details and IB to C2 conversion events
  • Opened by clicking the status column of a C2 signal row
  • A status of the the selected C2 signal is displayed: Signal ID, Status, Date Posted, Date Emailed, Date Expired, Date Killed, Date Traded, Trade Price, Scaling Pct
  • A history log of the IB to C2 conversion status transitions is displayed: Event Date, Status, C2 Signal, Description
