Install Procedure

  • After downloading hpbsignals.zip file, unzip it into a folder of your choice - for example home folder
  • No additional installation tasks are needed, all files reside in hpbsignals folder
  • Keep the original hpbsignals.zip files in a safe place - possibly a backup CD/DVD in case of reinstallation is needed
  • Before starting the application you must read (open in a browser) and agree to End User License Agreement located in license.html
