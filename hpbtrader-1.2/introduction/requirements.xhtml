Requirements

  • You need to have Interactive Brokers trading account with TWS or IB Gateway running
  • Make sure that IB is configured to accept socket connections: 'Edit-->Global Configuration-->API-->Settings-->Enable ActiveX and Socket Clients'
  • Make sure that IB setting 'Edit-->Global Configuration-->API-->Settings-->Download open orders on connection' is checked
  • It is recommended to install the latest TWS or IB Gateway
  • Any operating system with Java 7.x installed is supported. You can download Java here
