Strategy

Performance View displays the performance data of the selected strategy. It contains several tabs for displaying strategy performance, strategy PL graph, strategy orders, trades, trades logs, trades PL graph, trading instrument chart with technical indicators and historical quotes.

Strategy tab of the performance view displays the strategy performance metrics along with the strategy parameters, mode and status. Strategy logs table displays a series of snapshots of the strategy state. A new log entry is added when a new strategy or order event takes place. A graphical representation of the strategy cumulative profit and loss is available for a quick check of the strategy overall performance.

Strategy and strategy logs table columns:
  • Log Date: date of the log entry
  • Strategy: trading symbol ticker, interval, strategy name
  • Mode: IB - live order sent to IB Gateway, SIM - HPB Trader simulated order for testing and simulation
  • Act: checked if the strategy is currently active
  • Quant: default trading quantity
  • Params: strategy parameters
  • Pos: strategy position - current or at the time of the log entry creation
  • Trd-Ord-Fil: the number of all trades, all orders, all filled orders
  • Cls-Lon-Sho: the number of all closed trades, all long trades, all short trades
  • Win-Los: the number of winning trades, losing trades
  • PL Cum: cumulative profit/loss
