Manage Strategies

Multiple strategies can be defined for the selected series and one of the defined strategies can be marked active. This operation is performed by clicking the 'tool' button (toolbar button 11) and opening a dialog for managing the strategies. A new strategy must first be added by defining the strategy parameters. Subsequently its parameters like mode, default quantity, position and the strategy logic parameters can be edited and updated. If not needed any more the strategy can be deleted, but some caution is needed, since all related strategy data like orders and trades get deleted as well.

Manage Strategies table columns:
  • Strategy: ticker symbol, interval, trading strategy name
  • Act: checked if the strategy is active
  • Mode: IB - live order sent to IB Gateway, SIM - HPB Trader simulated order for testing and simulation
  • Position: strategy position
  • TQuant: trading quantity
  • Params: strategy logic parameters

Edit Strategy dialog fields:
  • Strategy: a name of the selected strategy
  • Active: is the strategy active
  • Mode: IB - live order sent to IB Gateway, SIM - HPB Trader simulated order for testing and simulation
  • Quantity: default trading quantity
  • Position: strategy position, can be updated if needed
  • Params: strategy logic parameters
  • Activate: make the selected strategy active
  • Update: perform the update action

Delete Strategy dialog fields:
  • Strategy: a name of the selected strategy
  • Active: is the strategy active
  • Mode: IB - live order sent to IB Gateway, SIM - HPB Trader simulated order for testing and simulation
  • Quantity: default trading quantity
  • Position: current strategy position
  • Params: strategy logic parameters
  • Delete: delete the selected strategy

New Strategy dialog fields:
  • Strategy: a name of the selected strategy
  • Active: is the strategy active
  • Mode: IB - live order sent to IB Gateway, SIM - HPB Trader simulated order for testing and simulation
  • Quantity: default trading quantity
  • Position: initial strategy position
  • Params: strategy logic parameters
  • Add: add a newly defined strategy
