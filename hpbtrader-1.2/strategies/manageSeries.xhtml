Manage Series

A new series is added by clicking the 'plus' button (toolbar button 6) and opening a dialog for entering the series parametrs. The series can also be deleted by clicking the 'delete' button (toolbar button 9). All corresponding strategies data, trades, orders, logs and other data related to the series also get deleted, so this operation needs to be used with caution. After the series is defined and enabled, the collection of the series quote data starts automatically and is visually presented by the series chart or quote table. Technical indicators like Exponential Moving Averages, MACD or Stochastics get calculated on demand when needed in the strategy or in the chart. Multiple automated strategies can be defined for the series and one of the defined strategy can be marked active. In addition to automated strategy orders, a manual order for the selected series and the active strategy can be submitted as well. This is performed by clicking the 'circle-plus' button (toolbar button 15) and opening a dialog for entering the manual order parameters.

Add New Series dialog fields:
  • Underlying: underlyng symbol of the financial instrument
  • Symbol: financial instrument symbol
  • Currency: currency of the financial instrument
  • Exchange: an exchange where the financial instrument is traded
  • Interval: series interval, for example 5 min, 60 min
  • Security Type: security type, STK - stock, OPT - option, FUT - future, FOP - future option, CSH - cash
  • Strategy Type: a name of the default strategy
  • Submit: add a newly defined series
  • Strategy Type: cancel the operation

Manual Order dialog fields:
  • Action: underlyng symbol of the financial instrument
  • Quantity: trading quantity
  • Symbol: financial instrument symbol
  • Currency: currency of the financial instrument
  • Sec Type: security type, STK - stock, OPT - option, FUT - future, FOP - future option, CSH - cash
  • OrderType: MKT - market order, LMT - limit order, STP - stop order
  • SubmitType: manual for the manual order
  • Initial Stop Pct: initial stop percent in case of an opening order
  • Target Pct: target percent in case of an opening order
  • Submit: submit the manual order (live or simulated according to the active strategy mode)
  • Strategy Type: cancel the operation
