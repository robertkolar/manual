Recent Orders

Recent orders table contains a list of all recent orders sent to either IB Gateway for execution or HPB Trader simulated orders. All orders that belong to either active trades or orders that are less than 2 days old are being shown.

Recent orders table columns:
  • Strategy: ticker symbol, interval, trading strategy name
  • Date Created: order creation date
  • Mode: IB - live order sent to IB Gateway, SIM - HPB Trader simulated order for testing and simulation
  • PermId: Interactive Brokers order id, unique across IB sessions, applies only to IB orders
  • OrdId: Interactive Brokers order id, not unique across IB sessions, applies only to IB orders
  • Trigger: a strategy defined trigger that initiated the creation of the order
  • Submit: auto - a strategy inititated automatic order, manual - manual order
  • Action: BTO - buy to open, BTC - buy to close, STO - sell to open, STC - sell to close, BREV- buy reversal, SREV - sell reversal
  • Quant: order trading quantity
  • Type: MKT - market order, LMT - limit order, STP - stop order
  • Limit: limit price in case of a limit order
  • Stop: stop price in case of a stop order
  • Fill Price: order fill price, if the order has already been filled
  • HBeat: heartbeat count, a special hearbeat algorithm for managing IB connection
  • Status: order status, one of: new, newRetry, submitReq, submitted, filled, cancelReq, canceled, unknown
