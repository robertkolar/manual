Real-time Quotes

Realtime quotes table displays a real time data from the live IB feed for the selected series.

Real-time quotes table columns:
  • Symbol: ticker symbol
  • SecT: security type, STK - stock, OPT - option, FUT - future, FOP - future option, CSH - cash
  • Bid Size: current bid size
  • Bid: current bid price
  • Ask: current ask price
  • Ask Size: current ask size
  • Last: last price
  • Last Size: last size
  • Volume: daily volume
  • Close: previous day close price
  • Change: percentage change
