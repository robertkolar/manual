Toolbar

Toolbar is an application's control panel containing buttons for triggering various actions such as IB gateway connection, adding/deleting series, strategy configuration, performance view, backtest, real-time quotes selection, manual order, chart view and so on.

Toolbar controls:
  • 1 - Refresh: trigger a manual refresh of all tables
  • 2 - Logout: logout from the application
  • 3 - Connect IB: connect to IB Gateway or Trader Workstation
  • 4 - Disconnect IB: disconnect from IB Gateway or Trader Workstation
  • 5 - Status IB: connection status, green - connected, red - disconnected
  • 6 - Add New Series: add a new series
  • 7 - Move Series Up: move the selected series row up
  • 8 - Move Series Down: move the selected series row down
  • 9 - Delete Series: delete the selected series and its related data
  • 10 - Show All Series: show both enabled and disabled series
  • 11 - Edit Strategies: add or remove strategies, change its parameters, designate an active strategy
  • 12 - Enable/Disable Series: enable or disable the selected series
  • 13 - Display RT Quotes: toggle a real-time quotes display for the selected series
  • 14 - Manual Backfill: trigger a manual backfill for the selected series
  • 15 - Enter Manual Order: enter a new manual order for the selected series
  • 16 - View Last Trade: view the last active trade details for the selected series
  • 17 - View Instrument Chart: open a chart window for the selected series
  • 18 - Open Performance View: open a performance view for the selected series
  • 19 - Select Strategy: select the strategy of the selected series to be shown in the performance view
  • 20 - Backtest Strategy: trigger a backtest of the sleected strategy and display the results in a backtest performance view
  • 21 - Backtest Start Date: choose a start date for the backtest
  • 22 - Backtest End Date: choose an end date for the backtest
  • 23 - Backtest All: perform a backtest on all available data (quotes) of the selected series and strategy
