Series

Series table displays basic information of the currently active or inactive series, active strategies, consolidated statistics and statuses of the currently active trades.

Series table table columns:
  • #: series display order ID
  • Undl: underlying symbol
  • Symbol: ticker symbol
  • Cur: instrument currency
  • Int: series interval - 5 min, 60 min
  • SecT: security type, STK - stock, OPT - option, FUT - future, FOP - future option, CSH - cash
  • Exch: an exchange on which the security is traded
  • Enb: series enabled or disabled
  • Rtq: are realtime quotes for the series enabled or disabled
  • Qts-Strs: the number of historical quotes and strategies
  • Last: last price
  • Act Str: currently acive strategy
  • Mode: IB - live order sent to IB Gateway, SIM - HPB Trader simulated order for testing and simulation
  • Pos: current position
  • Trd-Ord-Fil: the number of all trades, all orders, all filled orders
  • Cum PL: cumulative profit/loss
  • Typ: trade type - L (long) or S (short)
  • Unrl PL: unrealized profit/loss
  • Sts: active trade status
